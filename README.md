# Distributed_Randomization_WirelessHART

Distributed schedule randomization implementation on the GISOO simulator (https://github.com/araujokth/kth-gisoo). The codes contain the physical models and the controllers in Simulink Release 13 and the wirelessHART network in the Cooja simulator. I used Simulink in Matlab Release 13 as GISOO executable was written on a Real Time package which got renamed from Matlab Release 14 onwards. Hence for this setup I used GISOO, Matlab Release 13, Cooja and TinyOS to generate the executable for the TelosB motes.
